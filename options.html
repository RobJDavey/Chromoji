Emoji
Only emoji that iOS renders will be shown by default. This means that emoji will not be shown for symbols such as ©, ®, and ™.

A list of any domains which should not display emoji. This should be the last part of any domain seperated with commas "," e.g. google.com will include images.google.com, le.com will include google.com too as it ends in le.com (so be careful).
